# 📒 TallyOpen - Professional Accounting Made Simple

A high-performance, open-source accounting application designed for small businesses, shopkeepers, and professionals. TallyOpen combines the power of professional accounting with a clean, modern interface that anyone can master in minutes.

---

## 🚀 Standalone "Click-to-Run" Mode

TallyOpen is a fully portable, zero-dependency Windows desktop application. 

### 1. How to Start
- Double-click the **`Launch TallyOpen`** shortcut in the main folder.
- The application starts its own local server and automatically opens the dashboard in your browser.
- **Shutdown**: To close the app safely, use the **"Shutdown App"** button in the **Settings** menu.

### 2. Zero Installation
- **No Python Required**: Everything is bundled into the executable.
- **No Database Setup**: Uses a high-speed local SQLite engine (`tally.db`).
- **Completely Portable**: Move the entire folder to a USB drive or another PC, and it just works.

### 3. Default Login
- **Username**: `admin`
- **Password**: `admin123`
- *Note: Change your password immediately in Settings → Security.*

## 💎 Premium Features

### 🧾 Professional GST Invoices
- **Compliance**: Generates detailed PDF invoices with **CGST** and **SGST** tax breakdowns.
- **Smart Data**: Automatically pulls business info, customer details, and item prices into the bill.
- **PDF Export**: One-click generation and download of professional tax invoices.

### 📧 Integrated Email System
- **Direct Send**: Send PDF bills to your customers directly from the app.
- **Gmail Ready**: Works with Gmail via **Google App Passwords** (Settings → Keys).
- **Instant Testing**: Built-in "Test Email System" button to verify your connection in seconds.

### 📦 Intelligent Inventory
- **Stock Guard**: Prevents negative stock! The app validates your inventory levels before every sale.
- **Real-time Stats**: Track total stock value and product counts on the dashboard.
- **Low Stock Alerts**: Visual indicators when your products are running low.

### 📊 Advanced Reporting
- **P&L Statements**: Comprehensive Profit and Loss reports.
- **Ledger Books**: Detailed transaction history for every account.
- **Data Export**: Download any report as an Excel file with one click.

---

## 🛡️ Data & Privacy

- **Local-Only Storage**: All your financial data stays on **YOUR computer** in the `tally.db` file.
- **Build-Proof Persistence**: Your database is stored safely in the parent directory of the app folder, ensuring your data is never lost during app updates or rebuilds.
- **Offline First**: The core accounting features work 100% offline.

---

## 📋 Quick Setup Checklist
1. **Business Info**: Set your Shop Name, Address, and GSTIN in **Settings**.
2. **Email**: Setup your Gmail and App Password in **Settings → Keys** to send bills.
3. **Inventory**: Add your products and current stock in **Stock / Inventory**.
4. **Billing**: Start creating professional bills in the **Make a Bill** section.
