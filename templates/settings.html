How to get App Password

Google requires a special 16-character code to let apps send emails. Follow these steps:

  1. Go to your Google Account (myaccount.google.com).
  2. Click on the Security tab on the left.
  3. Ensure 2-Step Verification is turned ON.
  4. Search for "App Passwords" in the search bar at the top.
  5. Enter a name (e.g., "TallyOpen") and click Create.
  6. A yellow box will show a 16-character code (e.g., abcd efgh ijkl mnop).
Copy that code and paste it into the "Gmail App Password" box here.
